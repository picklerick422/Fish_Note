# 调研：SpringNote（参考项目）功能分析

来源：https://github.com/Radiant303/SpringNote —— "懒人实习记录工具"。定位：不仅记录，更帮助整理、沉淀和回顾，通过 AI 自动生成日报/周报/月报，让记录变成可检索、可对话的个人知识资产。

## 核心功能
1. **首页工作台**：等级/收益游戏化元素、活跃热力图（GitHub 风格贡献格子）、快速输入框（写下想法，AI 自动整理为结构化内容）、今日摘要卡片（完成事项 / 问题记录 / 明日计划 三栏统计）。
2. **AI 智能生成**：快速输入一段碎碎念，AI 自动整理成结构化便签内容。
3. **便签编辑**：左侧笔记本列表（可按日报/周报/月报分类筛选 + 搜索），右侧 Markdown 源码编辑 + 渲染预览双栏，支持代码块高亮、AI 补全预测（Tab 接受）。
4. **回忆书对话**：类 ChatGPT 的对话式记忆检索（RAG），展示"深度思考"过程与工具调用（关键词搜索、读取日报），支持 Markdown 渲染回答。
5. **自动报告生成**：基于日报自动生成周报/月报总结。
6. **统计面板**：年度活跃热力图、记录数/总结数/Token 用量总览、时间范围筛选（全部/最近30天/上月/上季度/自定义）。
7. **设置**：偏好设置、AI 供应商配置、默认模型、快捷键、关于。

## 界面风格
- 浅色极简：白色/浅灰底、圆角大卡片、左侧窄图标导航栏、清晰的信息层级
- 数字用大号粗体展示（如收益 9471）
- 绿色作为活跃/正向强调色（热力图、指标）

## 本项目实现要点（Web 版，后续 Electron 封装）
- 纯前端 + localStorage 持久化（Electron 可直接打包，无服务器依赖）
- AI 层：可插拔 Provider 接口（默认内置本地模拟引擎演示全部 AI 功能，可在设置中切换 OpenAI 兼容 API：填 baseURL/apiKey/model 即可接真实大模型）
- 页面：首页工作台、便签（Markdown 编辑器）、回忆书 AI 对话、报告中心、统计、设置
